Glossary

Front End: HTML/CSS, React.js.
Data from Google spreadsheets API.

The Glossary is an alphabetical list of Computer Science terms. These terms display as just the title and when clicked, reveal the definition. They also display a link to any related term that, when clicked, brings the user to that new term and reveals the definition. The Glossary can also be filtered alphabetically using the navigation bar located at the top of the screen.

This project uses HTML/CSS and ReactJS on the front end. It uses the Google Spreadsheets API to fetch the terms.

View this repository on Github here.
